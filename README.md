# Gym Management System

A desktop application built using **Java Swing and MySQL** to manage gym members, payments, and schedules.

## Features

* Member login
* Admin dashboard
* Add / update / delete members
* Member payment tracking
* Notification system
* Member schedule management

## Technologies Used

* Java
* Java Swing
* MySQL
* JDBC

## Project Structure

* `src/gymmanagement/ui` – UI pages
* `src/gymmanagement/model` – Data models
* `src/gymmanagement/util` – Database connection
* `assets` – Images and icons

## How to Run

1. Import the project into Eclipse.
2. Start the MySQL server.
3. Update database credentials in `dbconnection.java`.
4. Run `main.java`.
